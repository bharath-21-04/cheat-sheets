Bear Cheatsheet

Note-taking app shortcuts (macOS/iOS)

Shortcuts
🔍
Contents
Notes Editing Formatting Tags & Search Navigation Export
📝

Notes

⌘N             New note
⌘⇧N            New note in new window
⌘⌫             Delete note (Trash)
⌘P             Pin/unpin note
⌘⇧A            Archive note
⌘D             Duplicate note
⌘⇧M            Move note to tag
⌘⇧P            Print note

# Note info
⌘I             Show note info
(word count, created, modified)
✏️

Editing

⌘Z             Undo
⌘⇧Z            Redo
⌘C / ⌘V       Copy / Paste
⌘⌥V            Paste as plain text
⌘A             Select all
⌘⌥T            Insert todo
⌘K             Insert link
⌘⇧V            Insert file

# Fold/unfold
⌘⌥←            Fold heading
⌘⌥→            Unfold heading
🎨

Markdown Formatting

⌘B             Bold     **text**
⌘I             Italic   _text_
⌘U             Underline  ~text~
⌘⇧S            Strikethrough  -text-
⌘1-6           Heading 1-6  ## text
⌘L             Unordered list  - item
⌘⇧L            Ordered list    1. item
⌘⌥C            Code block  ```code```
⌘⇧E            Inline code  `code`
⌘'             Quote block  > text
⌘⇧U            Toggle markup visibility

# Special markers
::marker::     Highlight text
[x] / [ ]      Todo checkbox
🏷️

Tags & Search

# Tags (use # in note body)
#work          Simple tag
#work/project  Nested tag
#multi word#   Multi-word tag

# Search
⌘⇧F            Search all notes
⌘F             Search within note

# Search operators
@title(text)   Search in titles
@text(text)    Search in body
@tagged        Notes with tags
@untagged      Untagged notes
@todo          Notes with todos
@today         Modified today
@files         Notes with files
🧭

Navigation

⌘1             Show sidebar
⌘2             Show note list
⌘3             Show editor
⌘\             Toggle sidebar
⌘⌥↓            Next note
⌘⌥↑            Previous note
⌘Enter         Open note in new window

# Views
⌘⇧1            View by date
⌘⇧2            View by title
⌘⇧3            View by modification
📤

Export

# Export formats
Markdown (.md)
HTML
PDF
DOCX
JPEG (image)
TextBundle

# Export
⌘⇧S            Export note
File → Export Notes

# Import
File → Import Notes
Supports: Markdown, TextBundle,
Evernote, DayOne, other apps

# Sync
iCloud sync across devices
Bear Pro for all export formats
