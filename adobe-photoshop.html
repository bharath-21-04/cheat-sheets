Tool Shortcuts

V   Move                M   Marquee
L   Lasso               W   Quick Selection
C   Crop                I   Eyedropper
J   Healing Brush       S   Clone Stamp
B   Brush               E   Eraser
G   Gradient            O   Dodge/Burn
P   Pen                 T   Type
A   Path Selection      U   Shape
H   Hand                Z   Zoom
D   Default colors (B/W)
X   Swap foreground/background
Q   Quick Mask mode

# Cycle through hidden tools: Shift + shortcut
# e.g., Shift+L cycles Lasso tools
✂️

Selection

⌘A / Ctrl+A           Select all
⌘D / Ctrl+D           Deselect
⌘⇧I / Ctrl+Shift+I   Inverse selection
⌘⇧D / Ctrl+Shift+D   Reselect
⇧ + drag              Add to selection
⌥ + drag              Subtract from selection
⌘⌥R / Ctrl+Alt+R      Refine Edge / Select & Mask

# Transform selection
⌘T / Ctrl+T           Free Transform
⇧ drag corner         Constrain proportions
⌥ drag                Transform from center
Enter                  Apply transform
Esc                    Cancel transform
📚

Layers

⌘⇧N / Ctrl+Shift+N    New layer
⌘J / Ctrl+J           Duplicate layer (copy)
⌘⇧J / Ctrl+Shift+J   Cut to new layer
⌘E / Ctrl+E           Merge down
⌘⇧E / Ctrl+Shift+E   Merge visible
⌘G / Ctrl+G           Group layers
⌘⇧G / Ctrl+Shift+G   Ungroup
⌘[ / Ctrl+[           Send backward
⌘] / Ctrl+]           Bring forward
⌘⌥⇧E / Ctrl+Alt+Shift+E  Stamp visible

# Opacity: press 1-9, 0
# 5 = 50%   0 = 100%   15 (quick) = 15%

# Clipping mask
⌘⌥G / Ctrl+Alt+G
✏️

Edit

⌘Z / Ctrl+Z           Undo (multiple)
⌘⇧Z / Ctrl+Shift+Z   Redo
⌘⌥Z / Ctrl+Alt+Z      Step backward
⌘C / Ctrl+C           Copy
⌘V / Ctrl+V           Paste
⌘⇧V / Ctrl+Shift+V   Paste in place
⇧F5 / Shift+F5        Fill dialog
⌘⌫ / Ctrl+Backspace   Fill with bg color
⌥⌫ / Alt+Backspace    Fill with fg color
⌘⇧S / Ctrl+Shift+S    Save As
⌘⌥⇧S / Ctrl+Alt+Shift+S  Save for Web
👁️

View & Navigate

⌘+ / Ctrl++           Zoom in
⌘- / Ctrl+-           Zoom out
⌘0 / Ctrl+0           Fit to screen
⌘1 / Ctrl+1           Actual pixels (100%)
Space + drag           Pan
⌘R / Ctrl+R           Toggle rulers
⌘; / Ctrl+;           Toggle guides
⌘' / Ctrl+'           Toggle grid
F                      Cycle screen modes
Tab                    Hide/show panels
🖌️

Brushes & Paint

[ / ]                  Decrease / increase size
{ / }                  Decrease / increase hardness
,  / .                 Previous / next brush
< / >                  First / last brush

# Blend modes (while Brush active):
⇧⌥N  Normal            ⇧⌥M  Multiply
⇧⌥S  Screen             ⇧⌥O  Overlay
⇧⌥F  Soft Light         ⇧⌥H  Hard Light

# Flow vs Opacity
Opacity: amount per stroke
Flow: amount per stamp (builds up)
🎛️

Filters & Adjustments

# Quick adjustments
⌘L / Ctrl+L           Levels
⌘M / Ctrl+M           Curves
⌘U / Ctrl+U           Hue/Saturation
⌘B / Ctrl+B           Color Balance
⌘⇧U / Ctrl+Shift+U   Desaturate
⌘I / Ctrl+I           Invert

# Repeat last filter
⌘F / Ctrl+F

# Camera Raw
⌘⇧A / Ctrl+Shift+A

# Liquify
⌘⇧X / Ctrl+Shift+X
