Connection

# Supported databases
PostgreSQL   MySQL        MariaDB
SQLite       SQL Server   Oracle
MongoDB      Redis        Cassandra
CockroachDB  DuckDB       Snowflake

# New connection
⌘N             New connection
Fill: Host, Port, User, Password, DB

# Connection methods
Direct, SSH Tunnel, TLS/SSL
Connection string / URL

# Example URL
postgresql://user:pass@localhost:5432/db
mysql://user:pass@localhost:3306/db
⌨️

Keyboard Shortcuts

⌘N             New connection
⌘T             New tab
⌘W             Close tab
⌘⇧T            Reopen closed tab
⌘K             Open quick switch
⌘P             Command palette
⌘E             Open SQL editor
⌘R             Refresh data
⌘S             Save changes (commit)
⌘Z             Discard changes
⌘F             Filter/search
⌘G             Group by
⌘⇧S            Save as SQL file
⌘1-9           Switch tabs
⇧⌘[  / ⇧⌘]    Previous/next tab
💻

Query Editor

⌘E             Open SQL editor
⌘Enter         Run query
⌘⇧Enter        Run selected query
⌘;             Run current statement
⌘.             Stop query
⌘/             Toggle comment
⌘⇧L            Lowercase
⌘⇧U            Uppercase
⌘⌥F            Format SQL

# Multi-cursor
⌥ click        Add cursor
⌘D             Select next occurrence

# Autocomplete
Table names, columns, functions
Triggered automatically or Tab
📊

Data Editing

# Edit cell
Double-click   Edit cell
Enter          Confirm edit
Esc            Cancel edit
⌘↩             Edit multi-line

# Row operations
⌘⌥N            Insert new row
⌘⌫             Delete row
⌘C             Copy cell/row
⌘⇧C            Copy as INSERT

# Changes are staged (yellow)
⌘S             Commit changes
⌘Z             Discard all changes

# NULL values
⌃N / Ctrl+N    Set to NULL

# Sort
Click column header to sort
🧭

Navigation

⌘K             Quick switch (table/view)
⌘1-9           Switch to tab N
⌘⌥←/→          Collapse/expand sidebar

# Sidebar
Tables, Views, Functions, Procedures
Double-click → Open
Right-click → DDL / Duplicate / Drop

# Table structure
⌘⌥S            Show table structure
Columns, indexes, foreign keys

# Filter toolbar
⌘F             Open filter
column = value, IS NULL, LIKE, IN
📤

Import & Export

# Export data
Right-click table → Export
Formats: CSV, JSON, SQL, XML

# Export query result
Run query → Export button

# Import
File → Import → From CSV/SQL
Drag & drop CSV file

# Database dump
Right-click DB → Dump
mysqldump / pg_dump format

# Copy as
⌘⇧C            Copy as INSERT
Also: JSON, CSV, TSV, Markdown
