Tools (macOS / Windows)

V              Move tool
K              Scale tool
F              Frame tool
S              Slice tool
R              Rectangle
O              Ellipse
L              Line
P              Pen tool
T              Text tool
I              Color picker
C              Comment
H              Hand tool (pan)

Hold Space     Pan (temp hand tool)
📦

Layers & Objects

⌘G / Ctrl+G           Group selection
⌘⇧G / Ctrl+Shift+G   Ungroup
⌘[ / Ctrl+[           Send backward
⌘] / Ctrl+]           Bring forward
⌘⌥[ / Ctrl+Alt+[      Send to back
⌘⌥] / Ctrl+Alt+]      Bring to front
⌘⇧H / Ctrl+Shift+H   Show/hide selection
⌘⇧L / Ctrl+Shift+L   Lock/unlock
⌘R / Ctrl+R           Rename layer
Tab                    Select next sibling
Enter                  Select child
Escape                 Select parent
✏️

Edit & Transform

⌘D / Ctrl+D           Duplicate
⌘C / Ctrl+C           Copy
⌘V / Ctrl+V           Paste
⌘⇧V / Ctrl+Shift+V   Paste in place
⌘X / Ctrl+X           Cut
⌘Z / Ctrl+Z           Undo
⌘⇧Z / Ctrl+Shift+Z   Redo
⌘A / Ctrl+A           Select all

# Transform
⇧ drag                Constrain proportions
⌥ drag / Alt drag     Resize from center
⌥ hover               Show distance
Arrow keys             Move 1px
⇧ Arrow               Move 10px

# Align
⌥A / Alt+A            Align left
⌥D / Alt+D            Align right
⌥V / Alt+V            Align vertical center
⌥H / Alt+H            Align horizontal center
👁️

View & Zoom

⌘+ / Ctrl++           Zoom in
⌘- / Ctrl+-           Zoom out
⌘0 / Ctrl+0           Zoom to 100%
⌘1 / Ctrl+1           Zoom to fit
⌘2 / Ctrl+2           Zoom to selection
Z + click              Zoom in
Z + ⌥ click            Zoom out
⌘⇧3                   Toggle grid
⌘. / Ctrl+.           Toggle UI
⌘\ / Ctrl+\           Toggle panels
⌘⌥L / Ctrl+Alt+L      Toggle layers panel
🔤

Text

⌘B / Ctrl+B           Bold
⌘I / Ctrl+I           Italic
⌘U / Ctrl+U           Underline
⌘⇧< / Ctrl+Shift+<   Decrease font size
⌘⇧> / Ctrl+Shift+>   Increase font size
⌥⌘L / Ctrl+Alt+L      Align left
⌥⌘T / Ctrl+Alt+T      Align center
⌥⌘R / Ctrl+Alt+R      Align right
⌥⌘J / Ctrl+Alt+J      Justify

# Line/letter spacing
⌥⌘< / Ctrl+Alt+<      Decrease line height
⌥⌘> / Ctrl+Alt+>      Increase line height
🧩

Components

⌘⌥K / Ctrl+Alt+K      Create component
⌘⌥B / Ctrl+Alt+B      Detach instance
⌘⌥O / Ctrl+Alt+O      Open component

# Styles
⌘⌥C / Ctrl+Alt+C      Copy properties
⌘⌥V / Ctrl+Alt+V      Paste properties

# Boolean operations (select 2+ shapes)
Union, Subtract, Intersect, Exclude
Available in toolbar when shapes selected

# Auto Layout
⇧A / Shift+A          Add auto layout
⌘⇧A / Ctrl+Shift+A    Remove auto layout
🔗

Prototyping

# Switch to prototype mode
Tab between Design / Prototype / Inspect

# Presentation
⌘⌥Enter               Present from frame
⌘Enter                 Present from start

# In presentation
→ / ←                  Next / prev frame
R                      Restart
F                      Toggle flow list

# Export
⌘⇧E / Ctrl+Shift+E    Export selection
Formats: PNG, JPG, SVG, PDF
Scales: 1x, 2x, 3x, 4x
