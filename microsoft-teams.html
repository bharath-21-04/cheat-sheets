General Shortcuts

⌘/ / Ctrl+/           Show shortcuts
⌘E / Ctrl+E           Search
⌘. / Ctrl+.           Open command box
⌘, / Ctrl+,           Settings
⌘⇧M / Ctrl+Shift+M   Toggle mute
⌘⇧O / Ctrl+Shift+O   Toggle camera
⌘N / Ctrl+N           New chat
⌘⇧F5 / Ctrl+Shift+F5 Toggle full screen
Esc                    Close dialog/panel
📹

Meeting Shortcuts

⌘⇧M / Ctrl+Shift+M   Mute/unmute
⌘⇧O / Ctrl+Shift+O   Camera on/off
⌘⇧E / Ctrl+Shift+E   Screen share
⌘⇧B / Ctrl+Shift+B   Toggle blur
⌘⇧K / Ctrl+Shift+K   Raise hand
⌘⇧Space              Push-to-talk (muted)
⌘⌥L / Ctrl+Alt+L     Lobby
⌘⇧H / Ctrl+Shift+H   Hang up
⌘⇧A / Ctrl+Shift+A   Accept call
⌘⇧D / Ctrl+Shift+D   Decline call

# During share
⌘⇧Space              Give/take control
💬

Chat & Messages

⌘N / Ctrl+N           New chat
⌘⇧C / Ctrl+Shift+C   Go to compose
⌘⇧X / Ctrl+Shift+X   Expand compose
⇧Enter               New line in message
Enter                  Send message
⌘⇧F / Ctrl+Shift+F   Attach file
⌘⇧P / Ctrl+Shift+P   Schedule message

# Reactions (hover message)
Hover → click emoji button
Right-click → more options

# Search in chat
⌘F / Ctrl+F           Find in chat
🧭

Navigation

⌘1 / Ctrl+1           Activity

📞

Calls

⌘⇧A / Ctrl+Shift+A   Accept incoming
⌘⇧D / Ctrl+Shift+D   Decline incoming
⌘⇧C / Ctrl+Shift+C   Start audio call
⌘⇧U / Ctrl+Shift+U   Start video call
⌘⇧T / Ctrl+Shift+T   Transfer call
⌘⇧H / Ctrl+Shift+H   End call

# Dial pad
⌘⇧K / Ctrl+Shift+K   Toggle dial pad
🎨

Message Formatting

/commands:
/available                Set available
/busy                     Set busy
/dnd                      Set DND
/away                     Set away
/call [name]              Call someone
/chat [name]              Message someone

# Markdown in messages:
**bold**                  Bold
_italic_                  Italic
~~strikethrough~~         Strikethrough
`code`                    Inline code
```code block```          Code block
- list item               Bulleted list
1. list item              Numbered list
> quote                   Quote
---                       Horizontal line
