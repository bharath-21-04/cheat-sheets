Navigation

?              Show shortcuts help
G H            Home timeline
G E            Explore
G N            Notifications
G M            Messages
G P            Profile
G L            Likes
G S            Settings
G D            Display settings
G U            Go to user (type name)

# Close/back
Esc            Close dialog
📝

Compose

N              New post (tweet)
⌘Enter         Send post

# In compose box
@username      Mention user
#hashtag       Add hashtag
$TICKER        Stock cashtag

# Media
Attach: images (4), GIF, video, poll
Alt text: click image → Add description

# Thread
Click + to add next tweet in thread
Write all → Post all

# Schedule
Calendar icon → pick date/time
📜

Timeline Actions

J / K          Next / previous post
Enter          Open post details
. (period)     Load new posts
/              Search

# On selected post:
L              Like
R              Reply
T              Repost (retweet)
S              Share
B              Bookmark
U              Mute account
X              Block account
I              Open/close details panel
⌨️

🔍

Search Operators

from:username         From user
to:username           Replies to user
@username             Mentioning user
"exact phrase"        Exact match
word1 OR word2        Either word
-word                 Exclude word
#hashtag              Hashtag
since:2024-01-01      After date
until:2024-12-31      Before date
filter:links          With links
filter:images         With images
filter:videos         With video
min_retweets:100      Min reposts
min_faves:500         Min likes
min_replies:50        Min replies
lang:en               Language

# Combine:
from:elonmusk since:2024-01-01 min_faves:1000
