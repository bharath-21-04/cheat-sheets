Browser keyboard shortcuts

Shortcuts
🔍
Contents

Navigation

⌘L / Ctrl+L           Focus address bar
⌘← / Alt+←            Back
⌘→ / Alt+→            Forward
⌘R / Ctrl+R           Reload
⌘⇧R / Ctrl+Shift+R   Hard reload (skip cache)
⌘. / Esc              Stop loading
⌘D / Ctrl+D           Bookmark page
⌘⇧B / Ctrl+Shift+B   Show/hide bookmarks bar
⌘⇧O / Ctrl+Shift+O   Bookmarks manager
⌘H / Ctrl+H           Show history
⌘J / Ctrl+J           Downloads
📑

Tabs

⌘T / Ctrl+T           New tab
⌘W / Ctrl+W           Close tab
⌘⇧T / Ctrl+Shift+T   Reopen closed tab
⌘⌥→ / Ctrl+Tab        Next tab
⌘⌥← / Ctrl+Shift+Tab  Previous tab
⌘1-8 / Ctrl+1-8       Go to tab 1-8
⌘9 / Ctrl+9           Go to last tab
⌘⇧N / Ctrl+Shift+P   New private window
⌘N / Ctrl+N           New window

# Pin tab: Right-click → Pin Tab
# Mute tab: Right-click → Mute Tab
📄

Page Navigation

Space / ⇧Space        Scroll down / up
Home / End             Top / bottom of page
⌘↑ / Ctrl+Home        Top of page
⌘↓ / Ctrl+End         Bottom of page
⌘+ / Ctrl++           Zoom in
⌘- / Ctrl+-           Zoom out
⌘0 / Ctrl+0           Reset zoom
⌘P / Ctrl+P           Print
⌘S / Ctrl+S           Save page
⌘U / Ctrl+U           View source
F11                    Full screen
F7                     Caret browsing
🔍

Find & Search

⌘F / Ctrl+F           Find in page
⌘G / Ctrl+G           Find next
⌘⇧G / Ctrl+Shift+G   Find previous
/                      Quick find
Esc                    Close find

# Address bar search
*                      Search bookmarks
^                      Search history
%                      Search open tabs
#                      Search page titles
@engine term           Search with engine
🛠️

Developer Tools

⌘⌥I / F12             Toggle DevTools
⌘⌥C / Ctrl+Shift+C   Inspector (pick element)
⌘⌥K / Ctrl+Shift+K   Console
⌘⌥E / Ctrl+Shift+E   Network
⌘⌥J / Ctrl+Shift+J   Browser console
⌘⌥S / Ctrl+Shift+S   Debugger
⌘⌥M / Ctrl+Shift+M   Responsive design mode
⇧F5                   Performance profiler
⇧F7                   Style editor

# In DevTools
F5                     Reload (cache)
⌘⇧F / Ctrl+Shift+F   Search across files
📍

Address Bar

⌘L / Ctrl+L           Focus address bar
⌘↩ / Ctrl+Enter       Add .com + go
⇧↩ / Shift+Enter      Add .net + go
⌘⇧↩ / Ctrl+Shift+Enter  Add .org + go
⌥↩ / Alt+Enter        Open in new tab
⌘K / Ctrl+K           Focus search bar

# about: pages
about:config           Advanced settings
about:performance      Tab performance
about:memory           Memory usage
about:support          Troubleshooting info
about:crashes          Crash reports
🔧

Miscellaneous

⌘⇧Delete / Ctrl+Shift+Del  Clear history
⌘⇧A / Ctrl+Shift+A        Add-ons manager
⌘⇧S / Ctrl+Shift+S        Screenshot tool

# Reader mode
F9 or ⌘⌥R                  Toggle reader view

# Pocket
Click Pocket icon to save

# Container tabs (Multi-Account)
Long-press + → Choose container

# Picture-in-Picture
Hover video → click PiP icon
