Tools

V              Select
R              Rectangle
E              Ellipse
Y              Polygon
L              Line
P              Pen
T              Text
A              Artboard
I              Eyedropper
Space + drag   Pan

# Drawing
⇧ drag         Constrain shape
⌥ drag         Draw from center
Double-click   Edit path points
📦

Layers & Artboards

⌘G / Ctrl+G           Group
⌘⇧G / Ctrl+Shift+G   Ungroup
⌘L / Ctrl+L           Lock/Unlock
⌘; / Ctrl+;           Hide/Show
⌘K / Ctrl+K           Make component

# Artboards
A                      Artboard tool
Presets: iPhone, iPad, Web, Custom
⌘D / Ctrl+D           Duplicate artboard

# Z-order
⌘[ / Ctrl+[           Send backward
⌘] / Ctrl+]           Bring forward
⌘⇧[ / Ctrl+Shift+[   Send to back
⌘⇧] / Ctrl+Shift+]   Bring to front
✏️

Edit & Arrange

⌘D / Ctrl+D           Duplicate
⌘⇧R / Ctrl+Shift+R   Repeat grid
⌘Z / Ctrl+Z           Undo
⌘⇧Z / Ctrl+Shift+Z   Redo
⌘A / Ctrl+A           Select all
⌥ drag                Duplicate (drag copy)
Arrow keys             Nudge 1px
⇧ Arrow               Nudge 10px

# Boolean operations
Add (Union)            Combine shapes
Subtract               Cut shape
Intersect              Overlap only
Exclude Overlap        Remove overlap

# Responsive resize
Select element → Responsive Resize panel
👁️

View

⌘+ / Ctrl++           Zoom in
⌘- / Ctrl+-           Zoom out
⌘0 / Ctrl+0           Zoom to fit
⌘1 / Ctrl+1           100%
⌘3 / Ctrl+3           Zoom to selection
⌘' / Ctrl+'           Toggle grid
⌘⇧' / Ctrl+Shift+'   Toggle layout grid
⌘Y / Ctrl+Y           Toggle layers panel
⌘⇧Y / Ctrl+Shift+Y   Toggle assets panel
🔤

Text

⌘B / Ctrl+B           Bold
⌘I / Ctrl+I           Italic
⌘⇧< / Ctrl+Shift+<   Decrease font
⌘⇧> / Ctrl+Shift+>   Increase font
⌘⇧L / Ctrl+Shift+L   Align left
⌘⇧C / Ctrl+Shift+C   Align center
⌘⇧R / Ctrl+Shift+R   Align right

# Point vs Area text
Click = point text (grows)
Drag = area text (wraps)

# Auto-height: double-click bottom handle
🔗

Prototype Mode

# Switch to Prototype mode
Click "Prototype" tab at top

# Create interaction
1. Select element
2. Drag blue arrow → target artboard
3. Set trigger: Tap, Drag, Time, Voice
4. Set action: Transition, Auto-Animate
5. Set easing: Ease In/Out, Bounce...

# Preview
⌘Enter / Ctrl+Enter   Desktop preview
Record preview for sharing

# Auto-Animate
Name layers same across artboards
XD interpolates position, size, etc.
📤

Export & Share

⌘E / Ctrl+E           Export selected
⌘⇧E / Ctrl+Shift+E   Export batch

# Export formats: PNG, SVG, JPG, PDF
# Scales: 1x, 2x, 3x (iOS / Android)

# Sharing
Share → Design Review (get link)
Share → Development (specs + CSS)
Share → Presentation (fullscreen)

# Design specs auto-generate:
- Colors, fonts, sizes
- CSS snippets
- Asset downloads
- Spacing measurements
