Navigation

J / ↓              Next item
K / ↑              Previous item
G A                Go to All
G G                Go to feed list
G T                Go to Today
G S                Go to Saved
G R                Go to Read Later
G H                Go to highlights

# Views
⇧V                 Toggle view mode
(Magazine / Cards / Title-only)
📰

Articles

Enter / O          Open article
V                  Open in new tab
S                  Save for later
M                  Mark as read
⇧A                 Mark all as read
X                  Dismiss
B                  Save to board
T                  Tag article
.                  Share menu
N                  Add note
H                  Highlight text

# Sharing
Twitter, LinkedIn, Buffer, Slack
Pocket, Evernote, OneNote
Email, copy link

Manage Feeds

# Add new feed
+ button          Add content
Paste RSS URL or search

# Organize
Create folders/categories
Drag feeds between folders
Rename: right-click → Rename

# Feed settings
Right-click feed → Settings
- Auto-mark as read
- Mute
- Unsubscribe

# OPML
Settings → Import/Export OPML
Backup feed subscriptions
