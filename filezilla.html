Connection

# Quick Connect bar (top)
Host:     sftp://example.com   or ftp://
Username: user
Password: pass
Port:     22 (SFTP) / 21 (FTP)

# Site Manager (⌘S / Ctrl+S)
File → Site Manager
  Protocol: SFTP / FTP / FTPS
  Host:     server.com
  Port:     22
  Logon:    Normal / Key file / Interactive

# Protocols:
FTP      Port 21   Unencrypted
FTPS     Port 990  FTP over TLS
SFTP     Port 22   SSH File Transfer
🧭

Navigation

# Panel layout
Left:   Local (your computer)
Right:  Remote (server)
Top:    Message log
Bottom: Transfer queue

# Navigate
Double-click  Enter directory
..            Parent directory
Address bar   Type path directly
Bookmarks     ⌘B / Ctrl+B

# View
F5                 Refresh
⌘F / Ctrl+F        Search
Hidden files: Server → Force showing hidden files
📤

File Transfer

# Transfer files
Double-click       Transfer/open file
Drag & drop       Local ↔ Remote
Right-click → Upload/Download

# Transfer modes
Auto               Detect binary/ASCII
Binary             Images, executables
ASCII              Text files

# Queue
Bottom panel shows transfers
Queued / Failed / Successful tabs
Right-click failed → Retry

# Simultaneous transfers
Edit → Settings → Transfers
Max simultaneous: 1-10
⌨️

Keyboard Shortcuts

⌘S / Ctrl+S           Site Manager
⌘N / Ctrl+N           New connection tab
⌘W / Ctrl+W           Close tab
⌘Q / Alt+F4           Quit
⌘R / F5               Refresh
⌘B / Ctrl+B           Bookmarks
⌘D / Ctrl+D           Disconnect
⌘T / Ctrl+T           New tab
⌘⇧P / Ctrl+Shift+P   Process queue
Tab                    Switch panels
Enter                  Transfer file
Delete                 Delete file
F2                     Rename
⚙️

Important Settings

# Edit → Settings

# Connection
Timeout:           20 sec
Retries:           5
Keepalive:         30 sec

# Transfers
Max transfers:     2
Limit download:    Unlimited
Limit upload:      Unlimited

# FTP → Active vs Passive
Passive recommended (behind NAT/firewall)

# Interface
File exists action:
  Ask / Overwrite / Skip / Resume
💡

Tips

# Directory comparison
View → Directory Comparison
Highlights different files

# Synchronized browsing
View → Synchronized Browsing
Navigate both sides together

# File filtering
View → Filename Filter
Show only specific file types

# Export sites
File → Export → Export Site Manager
Backup site connections

# CLI (fzsftp)
Scripting support for automation

# Speed limit
Transfer → Speed Limit
Cap bandwidth usage
