Navigation

⌘K / Ctrl+K           Quick switch (channel/DM)
⌘⇧K / Ctrl+Shift+K   DM browser
⌘⇧L / Ctrl+Shift+L   Channel browser
⌘⇧T / Ctrl+Shift+T   Threads panel
⌘⇧A / Ctrl+Shift+A   All unreads
⌘⇧S / Ctrl+Shift+S   Saved items
⌘. / Ctrl+.           Toggle right sidebar
⌘, / Ctrl+,           Preferences
⌘[ / Ctrl+[           History back
⌘] / Ctrl+]           History forward
⌘1-9 / Ctrl+1-9       Switch workspace
⌥↑ / Alt+↑            Previous channel
⌥↓ / Alt+↓            Next channel
⌥⇧↑ / Alt+Shift+↑    Prev unread channel
⌥⇧↓ / Alt+Shift+↓    Next unread channel
💬

Messages

⌘N / Ctrl+N           New message
⇧Enter               New line
Enter                  Send message
⌘⇧Enter              Create snippet
⌘U / Ctrl+U           Upload file
↑                      Edit last message
⌘⇧\ / Ctrl+Shift+\   React to last message
⌘⇧M / Ctrl+Shift+M   Mute channel
E                      Edit message (when selected)
R                      Add reaction (when selected)
T                      Open thread (when selected)
P                      Pin message (when selected)
S                      Share message (when selected)
A                      Save message (when selected)
Delete                 Delete message (when selected)
🎨

Message Formatting

*bold*                    Bold
_italic_                  Italic
~strikethrough~           Strikethrough
`inline code`             Inline code

# Code block (triple backticks)
``` code block ```

> blockquote              Quote
>>> multi-line quote
- bulleted list
1. numbered list
:emoji_name:              Emoji shortcode

# Mentions
@username                 Mention user
@here                     Online members
@channel                  All members
@everyone                 Everyone
#channel-name             Link channel

Slash Commands

/status :emoji: text     Set status
/active                   Set active
/away                     Set away
/dnd [time]               Do not disturb
/mute                     Mute channel
/unmute                   Unmute channel
/topic [text]             Set channel topic
/invite @user             Invite to channel
/remove @user             Remove from channel
/leave                    Leave channel
/rename [name]            Rename channel
/archive                  Archive channel
/collapse                 Collapse images
/expand                   Expand images
/remind @user [text] [time]  Set reminder
/shrug                    ¯\_(ツ)_/¯
/giphy [text]             Random GIF
/who                      List members
/msg @user [text]         Direct message
🔍

Search

# Search box: ⌘G / Ctrl+G or ⌘F / Ctrl+F

# Search modifiers:
from:@username            From a user
in:#channel               In a channel
in:@username              In a DM
has:link                  Contains links
has:reaction              Has reactions
has:star                  Is saved
has:pin                   Is pinned
before:2024-01-01         Before date
after:2024-01-01          After date
during:January            During month
on:2024-06-15             On date
is:thread                 In threads

# Combine:
from:@alice in:#dev has:link after:2024-03-01
📞

Huddles & Calls

⌘⇧H / Ctrl+Shift+H   Start huddle
M                      Toggle mute (in huddle)
V                      Toggle camera

# Huddle features:
Screen share, threads, reactions
Live captions, notes

# Slack Connect
DM/channel with external orgs

# Workflow Builder
Automate: messages, forms, approvals
